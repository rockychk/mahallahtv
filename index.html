Panduan Install Mahallah TV

Panduan Lengkap Install Aplikasi Mahallah TV

Ikuti langkah-langkah berikut untuk menginstall dan mengatur aplikasi Mahallah TV di perangkat Anda.

1. Untuk STB Android

  1. STB yang sudah di Root - Pastikan STB Android Anda sudah di-root untuk akses penuh.
  2. Setting waktu pada STB - Atur waktu sesuai dengan zona waktu lokasi Anda.
  3. Instal Aplikasi Mahallah TV - Install aplikasi Mahallah TV melalui file APK yang telah disediakan.
  4. Buka Aplikasinya - Setelah install, buka aplikasi dan akan muncul tampilan utama.
  5. Ubah data sesuai keinginan - Sesuaikan waktu Countdown dan pengaturan lainnya dengan menggunakan mouse.
Catatan Penting:

Paket 3 Bulanan: Gunakan mouse untuk mengubah semua pengaturan.

Paket Tahunan dan Lifetime: Sudah mendapatkan Remote Aplikasi Mahallah TV.

Untuk Paket Tahunan dan Lifetime (Remote Control):
  1. Aktifkan hotspot di HP - Aktifkan hotspot dari HP Anda (bisa juga melalui STB jika mendukung fitur Wi-Fi Hotspot).
  2. Cek IP Address di HP Anda - Periksa alamat IP yang terhubung.
  3. Akses Remote via Browser - Buka browser Chrome dan ketikkan IP dengan tambahan port 8080, contoh: 192.168.1.39:8080
  4. Masukkan PIN default - Gunakan PIN default: 1234
  5. Ganti semua data sesuai keinginan - Gunakan remote untuk mengatur semua pengaturan sesuai kebutuhan.

2. Untuk TV Android (Smart TV)

  1. Setting waktu sesuai lokasi - Atur waktu di TV Android sesuai dengan zona waktu lokasi Anda saat ini.
  2. Instal Aplikasi Mahallah TV - Install aplikasi melalui Google Play Store atau file APK.
  3. Buka Aplikasinya - Jalankan aplikasi untuk melihat tampilan utama.
  4. Ubah data sesuai keinginan - Sesuaikan pengaturan dengan menggunakan mouse atau remote TV.
Catatan Penting:

Paket 3 Bulanan: Gunakan mouse untuk mengubah semua pengaturan.

Paket Tahunan dan Lifetime: Sudah mendapatkan Remote Aplikasi Mahallah TV.

Untuk Paket Tahunan dan Lifetime (Remote Control):
  1. Aktifkan hotspot - Aktifkan hotspot dari HP atau melalui Android TV jika memungkinkan.
  2. Cek IP Address - Periksa alamat IP di HP atau TV Anda.
  3. Akses Remote via Browser - Buka browser Chrome dan ketikkan IP dengan port 8080, contoh: 192.168.1.39:8080
  4. Masukkan PIN default - Gunakan PIN default: 1234
  5. Ganti semua data sesuai keinginan - Atur semua pengaturan menggunakan remote aplikasi.

Tanya Jawab (FAQ)

Pertanyaan yang Sering Ditanyakan

Temukan jawaban untuk pertanyaan umum seputar Mahallah TV

1. Apa itu Mahallah TV?

Mahallah TV adalah aplikasi Adzan Otomatis & Informasi Masjid berbasis Android yang dirancang khusus untuk TV Masjid / STB, menampilkan jadwal shalat, adzan otomatis, pengumuman, dan media visual secara profesional.

Aplikasi ini dibuat khusus untuk kebutuhan masjid, musala, dan lembaga keislaman dengan tampilan yang elegan dan fitur lengkap.

2. Apakah Mahallah TV membutuhkan internet?

Tidak.

Mahallah TV dapat berjalan 100% offline. Semua fitur utama seperti:

  • Adzan otomatis
  • Audio shalawat
  • Tampilan informasi
  • Remote kontrol

tetap berfungsi tanpa koneksi internet setelah proses instalasi awal.

3. Bagaimana cara mengontrol Mahallah TV?

Mahallah TV dapat dikontrol melalui HP Android menggunakan remote berbasis browser (tanpa install aplikasi tambahan).

Cukup buka browser di HP Anda dan akses:

http://IP-TV:8080
contoh: http://192.168.1.30:8080

Anda akan mendapatkan antarmuka remote lengkap untuk mengatur semua fitur Mahallah TV.

4. Apakah HP dan TV harus satu jaringan WiFi?

Ya.

HP dan TV/STB harus berada di jaringan yang sama (WiFi lokal / hotspot).

Internet tidak wajib, yang penting HP dan TV berada dalam satu jaringan lokal yang sama untuk remote control berfungsi.

5. Perangkat apa saja yang didukung?

Mahallah TV mendukung berbagai perangkat Android:

Android TV STB Android (seperti HG680-P / HG860-P) TV Android Box Perangkat Android lainnya

Minimum Android: Android 5.0 (Lollipop)

6. Apakah adzan bisa diputar secara otomatis?

Bisa.

Adzan akan diputar otomatis sesuai jadwal shalat yang telah diatur, dan juga bisa:

  • Diputar manual dari remote
  • Diatur durasi iqamah (hitung mundur)
  • Disesuaikan volume dan audio
  • Dipilih suara adzan yang diinginkan
7. Apakah bisa menampilkan teks pengumuman masjid?

Bisa.

Mahallah TV mendukung berbagai fitur informasi:

  • Running text (teks berjalan)
  • Pesan pengumuman statis
  • Informasi kegiatan masjid
  • Ganti teks langsung dari HP via remote
  • Slide informasi dengan gambar
8. Bagaimana jika listrik atau TV mati?

Tenang

Semua pengaturan Mahallah TV tersimpan otomatis di perangkat. Saat TV hidup kembali, aplikasi akan:

  • Melanjutkan pengaturan terakhir
  • Siap diputar tanpa setting ulang
  • Menjalankan adzan sesuai jadwal
  • Menampilkan informasi yang sudah diatur
9. Apakah aman digunakan di masjid?

Aman & stabil.

Mahallah TV dirancang khusus untuk penggunaan di masjid:

  • Tidak membutuhkan akun login
  • Tidak mengirim data ke internet
  • Tidak ada iklan sama sekali
  • Bisa dikunci layar (kiosk mode)
  • Cocok untuk pemakaian 24 jam non-stop
  • Stabil dan tidak mudah crash
10. Apakah Mahallah TV cocok untuk semua masjid?

Sangat cocok, baik untuk:

  • Masjid besar
  • Musholla kecil
  • Sekolah dan pesantren
  • Aula dan ruang publik
  • Rumah pribadi

Mahallah TV dirancang sederhana, stabil, dan mudah digunakan oleh pengurus masjid tanpa perlu keahlian teknis khusus.
